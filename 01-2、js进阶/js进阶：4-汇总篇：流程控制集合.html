知识点汇总

  1. if
    单个是做判断（成，就执行。不成，就滚蛋！）
  2. if else
    一组是二选一（成，执行条件1，否则，执行另一个）
  3. if else
    多个是多重判断（多个条件看谁成，最后只执行成的那一个）
  4. switch
    是多种选择（相比多组if else，可以同时执行多个满足条件后 的函数）
  5. for
    是重复重复（遍历）
  6. while
    是反反复复
  7. do while
    是来来回回
  8. break
    是退出循环
  9. continue
    是跳过并继续循环
